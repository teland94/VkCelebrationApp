# VkCelebrationApp
Simple Web Application for finding and congratulating birthday persons in Vkontakte.

## Used APIs and Frameworks
  * ASP.NET Core 2.0 (API)
  * Vkontakte API for .NET
  * Telegram Bot API
  * Entity Framework Core 2.0 with SQL Server
  * Autofac
  * Automapper
  * Angular 5
  * Bootstrap 3
  * Azure Face API
